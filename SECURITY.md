# Security Policy

## Supported Versions

This project is in its early public releases. Security fixes will be applied to the latest published version.

## Reporting A Vulnerability

Please report suspected vulnerabilities privately to the maintainers instead of opening a public issue.

When possible, include:

- a description of the issue
- reproduction steps
- affected platforms and versions
- any proof-of-concept or logs that help confirm impact

We will review reports as quickly as possible and coordinate a fix before public disclosure when appropriate.
