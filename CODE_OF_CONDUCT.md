# Code of Conduct

## Our Pledge

We want `react-native-nitro-charts` to be a welcoming, respectful project for everyone.

Contributors, maintainers, and community members are expected to participate in ways that make collaboration constructive and safe.

## Expected Behavior

- Be respectful and assume good intent.
- Give feedback about code and ideas, not people.
- Welcome questions from contributors at all experience levels.
- Share context clearly when suggesting changes or reporting problems.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks
- Trolling, intimidation, or sustained disruption
- Publishing private information without permission
- Sexualized language or imagery in project spaces

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, pull requests, or other contributions that violate this Code of Conduct.

## Scope

This Code of Conduct applies to project spaces and to public spaces when someone is representing the project.

## Reporting

If you experience or witness unacceptable behavior, contact the maintainers privately.
